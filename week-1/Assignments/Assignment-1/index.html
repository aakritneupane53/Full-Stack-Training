Semantic HTML

Semantic HTML is the practise of using html elements/tags that clearly describe their meaning. Unlike the generics tags like 

<div>
and 
<span>
 which are just used for styling purposes.Semantic HTML improves accessibility, SEO, and code readability. It helps browsers and developers understand the structure and meaning of content.

Benefits of Using Semantic HTML

  • Accessibility

    Screen readers and other assistive technologies use semantic elements as landmarks to help users with visual impairments navigate and understand the page structure.

  • SEO (Search Engine Optimization)

    Search engine crawlers prioritize semantic elements to identify the most relevant parts of a page, such as the main article content versus navigation links, which can improve search rankings.
