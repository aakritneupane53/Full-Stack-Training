# Weather Application

A modern weather application built with Plain HTML CSS JS and Open Weather API that allows users to search for weather information for any city.


## Features

- **City Search**: Search weather information for any city worldwide
- **Real-time Weather Data**: Displays current weather conditions including temperature, humidity, and more

- **Error Handling**: Displays user-friendly error messages when searches fail
- **Responsive Design**: Clean UI for a better user experience

## Tech Stack

- **Frontend Framework**: Vanilla JS, HTML
- **Styling**: CSS


## Usage

1. Install dependencies:

   ```bash
   Setup Open Weather API KEY, Paste your open weather API key at the top and initialise to the variable apiKey
   ```

2. Open in your browser or use live server and navigate to `http://localhost:5000`

## Project Structure

```
src/
├── index.html             # Main application component
```

## How to Use

1. Enter a city name in the search form
2. Click search or press Enter
3. View the weather information for that city
4. Handle any errors gracefully with helpful messages
